Exam Scheduler

A desktop application for generating conflict-free exam timetables with automatic classroom assignment

Overview

Exam Scheduler is a JavaFX-based desktop application designed to generate conflict-free exam schedules while automatically assigning classrooms.

The system uses a two-phase scheduling architecture that separates time-slot decisions from classroom allocation, ensuring scalability and robustness.

Main dashboard with calendar-based exam schedule

Key Features

  • CSV import for students, courses, classrooms, and enrollments
  • Two-phase scheduling architecture
  • No student exam conflicts
  • Maximum 2 exams per day per student
  • Minimum 90-minute gap between exams
  • Automatic classroom assignment (bin-packing)
  • Room splitting for large courses
  • Student & classroom filtered views
  • CSV export of generated schedules
  • Stress-tested up to 10,000 students

Architecture

Phase 1: Time Slot Assignment

  • Student-centric, conflict-aware scheduling
  • Most constrained courses scheduled first
  • Bin-packing into existing time slots
  • Partial scheduling supported

Phase 2: Classroom Assignment

  • Resource-aware bin-packing
  • Largest classrooms assigned first
  • Automatic multi-room splitting
